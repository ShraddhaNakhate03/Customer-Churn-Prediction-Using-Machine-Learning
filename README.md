
# 📊 Customer Churn Prediction Using Machine Learning

## 📌 Project Overview

Customer churn is one of the biggest challenges faced by businesses. This project focuses on predicting whether a customer is likely to leave a company using Machine Learning techniques. By analyzing customer behavior and historical data, the model helps businesses take proactive actions to retain customers.

This is an **end-to-end Machine Learning project** that includes data preprocessing, exploratory data analysis, model building, evaluation, and prediction.

---

## 🎯 Project Objectives

* Analyze customer data to understand churn behavior
* Perform data cleaning and preprocessing
* Conduct Exploratory Data Analysis (EDA)
* Build and train machine learning models
* Evaluate model performance
* Predict customer churn to support business decision-making
